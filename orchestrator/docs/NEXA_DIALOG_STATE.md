# NEXA Dialog State (v0.1.5)

## Глобальные команды (перебивают ВСЁ)
- Расписание
- Аренда
- Администратор
- Возраст
- Отмена / Сброс

Правило:
Если сообщение пользователя матчится на глобальную команду — она обрабатывается первой,
независимо от sticky-интента и текущего шага сценария.

## Sticky-интент аренды
Правило:
Sticky "Аренда" держится только пока пользователь продолжает вводить параметры аренды.
Любая глобальная команда должна сбрасывать sticky или переключать режим.

## Детские группы
Правило:
Сценарий "Детские группы" подразумевает запись РЕБЁНКА. Внутри нельзя спрашивать "ребёнок/взрослый".

## Точки выдачи quick_actions
- Вопрос "Какое время удобно?" → выдавать quick_actions
- В аренде: длительность и формат (минимум quick_actions)
