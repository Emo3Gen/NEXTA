# NEXA Manual Tests (v0.1.5)

## Детские группы — happy path
1) "Хочу записать ребёнка"
2) возраст → интерес → бот показывает расписание
3) бот спрашивает "Какое время удобно?" и показывает quick_actions
4) выбрать слот кликом
5) телефон
6) подтверждение

Ожидаем:
- нет вопроса "ребёнок/взрослый"
- слот выбирается кликом

## Детские группы — отказ телефона
1) пройти до телефона
2) ответить "не дам"
Ожидаем:
- нет бесконечного цикла
- бот даёт понятную альтернативу/пояснение

## Аренда — happy path
1) "Хочу аренду"
2) дата/время/длительность/люди
Ожидаем:
- sticky аренды удерживает сценарий
- в конце "принято, передаю админу"

## Глобальные команды перебивают sticky
1) войти в аренду (sticky активен)
2) написать "Расписание"
Ожидаем:
- показывается расписание
- sticky не мешает

## Переключение на возраст ребёнка из аренды
1) войти в аренду
2) написать "Возраст 6"
Ожидаем:
- sticky аренды не перехватывает
- бот корректно реагирует на возраст как глобальную команду/переключение
